## 📖 About The Project

This project is a **full-stack Laravel application** designed to simulate a real-world financial management system.

It allows users to:

* Track income and expenses
* Categorize transactions
* Set monthly budgets
* Monitor financial health with summaries
* Export data for external use

The goal of this project is to strengthen **backend development skills** and demonstrate the ability to build a **complete, production-style Laravel application**.
